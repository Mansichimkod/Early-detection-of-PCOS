# 🧠 PCOS Prediction System with Explainable AI

## 📌 Overview
A Machine Learning-based web application that predicts the likelihood of Polycystic Ovary Syndrome (PCOS) using patient health data. The system integrates Explainable AI (SHAP) to provide transparency by showing how each feature contributes to the prediction.

---

## 🚀 Key Features
- Predicts PCOS using a trained classification model  
- Interactive web interface built with Flask  
- Explainable predictions using SHAP  
- Displays feature importance for better understanding  

---

## 🛠️ Tech Stack
Python, Flask, Scikit-learn, Pandas, NumPy, SHAP, HTML, CSS  

---

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies:  
   pip install -r requirements.txt  
3. Run the app:  
   python app.py  
4. Open in browser:  
   http://127.0.0.1:5000
